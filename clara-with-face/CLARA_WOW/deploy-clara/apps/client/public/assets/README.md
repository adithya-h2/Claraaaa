# Logo Assets

Place your SVIT logo image file here.

## Required File:
- **Filename:** `svit-logo.png`
- **Location:** `CLARA_CHILL/apps/client/public/assets/svit-logo.png`

## Instructions:
1. Place your cropped SVIT logo image (the circular logo with sun, book, and flame) in this folder
2. Name it exactly: `svit-logo.png`
3. The image should be a PNG file with transparent background (or the logo on a transparent background)
4. Recommended size: 400x400px or larger (the CSS will resize it automatically)

The logo will be displayed on the welcome/sleep screen of the application.
